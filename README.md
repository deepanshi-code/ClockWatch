A modern **Study Timer + Stopwatch Web App** built using **HTML, CSS, and JavaScript**.
Designed with a clean UI, glassmorphism effects, and interactive features to boost productivity.

---

## 🚀 Features

### ⏳ Timer (Countdown Mode)

* Set custom time (Days, Hours, Minutes, Seconds)
* Live countdown display
* Circular progress animation (SVG-based)
* End time calculation
* Alert + Notification + Sound on completion

### ⏱️ Stopwatch (Count-Up Mode)

* Start / Pause / Reset functionality
* Real-time time tracking (HH:MM:SS)
* Clean UI with no input required

### 🌗 Theme Switcher

* Dark ↔ Light mode toggle
* Smooth UI transitions

### 🔄 Mode Switch

* Toggle between **Timer** and **Stopwatch**
* Dynamic UI changes
* Title and layout updates automatically

### 🕒 Real-Time Clock

* Displays current system time
* Updates every second

---

## 🛠️ Technologies Used

* **HTML5** → Structure
* **CSS3** → Styling (Glassmorphism, Flexbox, Animations)
* **JavaScript (Vanilla)** → Logic & Interactivity
* **SVG** → Circular progress visualization

---

## 🧠 Core Concepts Used

* Event-driven programming
* DOM manipulation
* Time handling using seconds
* `setInterval()` for real-time updates
* CSS class toggling for theme & mode switching
* SVG stroke animation
